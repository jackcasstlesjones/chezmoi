# Ultra-Brevity Writing Style Skill

## Purpose

Rewrite any text at ~25% of original word count using caveman-style compression. Zero content loss — every idea survives, just stripped to bone.

## Core Rules

1. **Cut function words ruthlessly** — articles (a, the), conjunctions (and, but, so), auxiliary verbs (is, are, was, were, have, has), filler phrases (in order to, it is important to note that)
2. **Telegraphic grammar** — drop subjects when obvious, drop "to be" verbs, use nouns as verbs
3. **No hedging** — remove "somewhat", "generally", "it could be argued", "in many cases"
4. **No preamble** — never restate the question, never say "great question", never summarise what you're about to do
5. **Compress examples** — one example max per concept, no elaboration
6. **Active over passive** — always
7. **Numbers not words** — "3" not "three", "%" not "percent"

## Caveman Patterns

| Before                               | After           |
| ------------------------------------ | --------------- |
| It is important that you remember to | Remember:       |
| In order to achieve this goal        | To achieve this |
| The reason why this works is because | Works because   |
| You should make sure that you are    | Be              |
| There are a number of ways in which  | Ways to         |
| This is something that needs to      | Must            |

## What NOT to Remove

- Specific facts, figures, names, dates
- Technical terms (compress around them, not through them)
- Logical connectives that change meaning (however, unless, except)
- Any unique claim or insight

## Output Format

Rewrite in original file.
